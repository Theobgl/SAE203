# 🚀 DÉMARRAGE RAPIDE - MODE HYBRIDE

## ⚡ UNE SEULE COMMANDE POUR DÉMARRER

```bash
python initialize_hybrid.py && python manage.py runserver
```

Puis ouvrez: **http://localhost:8000** 🎉

---

## 🤔 QUI FONCTIONNE AVEC QUOI?

### ✅ MongoDB + Django
```bash
# 1. Démarrer MongoDB
net start MongoDB

# 2. Initialiser
python initialize_hybrid.py

# 3. Lancer
python manage.py runserver

# ✅ Résultat: MongoDB utilisé
```

### ✅ SQLite + Django (SANS MONGODB)
```bash
# 1. Pas besoin de MongoDB!

# 2. Initialiser
python initialize_hybrid.py

# 3. Lancer
python manage.py runserver

# ✅ Résultat: SQLite utilisé automatiquement
```

### ✅ Les deux fonctionnent!
L'app détecte automatiquement et bascule! 🔄

### Scénario 2: J'ai MongoDB d'instalé
```bash
net start MongoDB
python initialize_hybrid.py
python manage.py runserver
```
✅ Utilise MongoDB automatiquement

### Scénario 3: Je passe de MongoDB à SQLite
```bash
# Arrêtez MongoDB
net stop MongoDB

# Relancez l'app
python initialize_hybrid.py
python manage.py runserver
```
✅ Bascule automatiquement à SQLite

### Scénario 4: Je passe de SQLite à MongoDB
```bash
# Démarrez MongoDB
net start MongoDB

# Relancez l'app
python initialize_hybrid.py
python manage.py runserver
```
✅ Bascule automatiquement à MongoDB
