================================================================================
🗄️ STRUCTURE MONGODB
================================================================================

5 Collections MongoDB créées:

1. categories
   _id: ObjectId
   nom: String (unique)
   descriptif: String

2. produits
   _id: ObjectId
   nom: String
   prix: Float
   marque: String
   date_peremption: Date
   categorie: Reference→categories
   photo: FileField

3. clients
   _id: ObjectId
   numero_client: Integer (unique)
   nom: String
   prenom: String
   date_inscription: Date
   adresse: String

4. commandes
   _id: ObjectId
   numero_commande: Integer (unique)
   client: Reference→clients
   date_commande: DateTime

5. lignes_commande
   _id: ObjectId
   commande: Reference→commandes
   produit: Reference→produits
   quantite: Integer

🐛 DÉPANNAGE RAPIDE
================================================================================

❌ "ServerSelectionTimeoutError"
   → MongoDB n'est pas lancé
   → Lancer: net start MongoDB (Windows)

❌ "Port 8000 already in use"
   → Utiliser: python manage.py runserver 8001

❌ "TemplateDoesNotExist"
   → Vérifier dossier templates/ existe
   → Vérifier fichiers HTML présents

❌ "CSRF verification failed"
   → Rafraîchir page
   → Clair cache navigateur

❌ "No data displayed"
   → MongoDB running?
   → initialize_db.py executed?
   → Check Django logs
